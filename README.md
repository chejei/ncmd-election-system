# Election System (Laravel + React + Vite)

## Requirements

-   PHP 8.1+
-   Composer
-   Node.js 18+
-   NPM
-   MySQL

## Installation

1. Clone the repo
2. Run `composer install`
3. Run `npm install`
4. Copy `.env.example` to `.env` and update database config
5. Run migrations: `php artisan migrate`
6. Start dev server: `npm run dev`
